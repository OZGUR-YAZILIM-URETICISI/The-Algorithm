Temel Komutlar
KomutDönüşAçıklama & Örnek
attempt(s)boolŞifre denemesi. Doğruysa True, yanlışsa False. Sayaç artar.
if attempt("123456"): return "123456"
get_hint(pos)"low"/"high"pos. hane 0-4 → "low", 5-9 → "high". Arama uzayını yarıya indirir.
h = get_hint(0) → "low"
binary_search(pos)intO pozisyondaki rakamı max 4 denemede bulur. En verimli komut.
d = binary_search(2) → 7
get_freq(pos)list[int]O pozisyonda geçmişte en sık görülen rakamlar (sıralı). İstatistiksel öncelik.
freq = get_freq(0) → [3,7,1,...]
mark_tried(s)NoneŞifreyi "denendi" listesine ekle. skip_tried ile birlikte kullan.
mark_tried("000000")
skip_tried(s)boolDaha önce denendiyse True döner (atla). Tekrar denemeyi önler.
if skip_tried(guess): continue
get_tried_count()intBu oturumda toplam kaç şifre denendi.
n = get_tried_count()
Teknikler & Karmaşıklık
TeknikO()Açıklama
Kaba Kuvvet10^nTüm kombinasyonlar. ≤6 hane için kullanılabilir.
for i in range(10**n): attempt(str(i).zfill(n))
Hane Bazlın×10Her haneyi bağımsız bul, birleştir. Çok daha hızlı.
for pos in range(n): for d in range(10): test_pos(pos,d)
İkili Araman×4Her hane max 4 denemede. 100K hane = 400K deneme.
''.join(str(binary_search(i)) for i in range(n))
Frekans Önc.n×avgGeçmiş veriden en olası rakamı önce dene.
for d in get_freq(pos): if correct(pos,d): break
Hibritn×2-3get_hint + get_freq + skip_tried kombinasyonu. En hızlı.
hint=get_hint(p); [d for d in get_freq(p) if uygun(d,hint)]
Hata Kodları
KodAnlam
ERR_NO_TARGETŞifre üretilmemiş.
ERR_SYNTAXPython sözdizimi hatası. Kodu kontrol et.
ERR_NO_CRACK_FNcrack(n) fonksiyonu tanımlanmamış.
WARN_SLOWYavaş algoritma. Donanım yükselt veya daha iyi algo yaz.
WARN_REPEATAynı deneme tekrar yapıldı. skip_tried/mark_tried kullan.
OK_SOLVEDŞifre bulundu, ödül verildi.
Dosya & Depo Komutları
KomutDönüşAçıklama & Örnek
save_file(isim, içerik)Noneİçeriği Depo'ya kaydeder. Aynı isimde dosya varsa üzerine yazar.
save_file("sonuclar", "123 456 789")
append_file(isim, satır)NoneDosyaya yeni satır ekler. Yoksa oluşturur.
append_file("log", solved_pw)
read_file(isim)list[str]Dosyayı satır listesi olarak okur.
lines = read_file("sonuclar")
file_exists(isim)boolDosya var mı kontrol eder.
if file_exists("log"): ...
clear_file(isim)NoneDosya içeriğini siler (dosya kalır).
clear_file("gecici")
delete_file(isim)NoneDosyayı tamamen siler.
delete_file("eski_log")
open_folder(isim)NoneDosyayı Depo sekmesinde açar/gösterir. Koşullu açma için if ile kullan.
if count("tekrar") > 10: open_folder("tekrar")
count_file(isim)intDosyadaki satır sayısını döner.
n = count_file("log") → 42
Kütüphane Komutları
KomutDönüşAçıklama & Örnek
load_lib(isim)list[str]Kütüphane sekmesinden bir kütüphaneyi satır listesi olarak yükler.
passwords = load_lib("6hane_common")
search_lib(isim, s)boolKütüphanede s değeri var mı diye bakar.
if search_lib("common", guess): ...
lib_size(isim)intKütüphanedeki kayıt sayısını döner.
n = lib_size("6hane") → 1000
save_to_lib(isim, s)NoneKütüphaneye yeni girdi ekler (çalışma zamanında).
save_to_lib("bulunanlar", solved_pw)
Kontrol Akışı & Yardımcı Komutlar
KomutDönüşAçıklama & Örnek
if / elif / else—Standart Python koşulları. Tüm komutlarla kullanılabilir.
if count_file("log") > 10: open_folder("log")
for / while—Döngüler. Kütüphane üzerinde iterasyon için idealdir.
for pw in load_lib("liste"): if attempt(pw): return pw
log(mesaj)NoneKonsola mesaj yazdırır. Debug için kullan.
log("Pozisyon " + str(pos) + " bulundu")
get_progress()floatMevcut ilerleme yüzdesini döner (0.0 - 100.0).
if get_progress() > 50: log("Yarıya ulaştı")
sleep(ms)Nonems milisaniye bekler. Görsel takip için yavaşlatmada kullan.
sleep(100) # 0.1 saniye bekle
count(dosya)intcount_file ile aynı — kısayol.
if count("tekrar") >= 10: open_folder("tekrar")
get_target_len()intHedef şifre uzunluğunu döner. get_length() ile aynı.
n = get_target_len()
